Текстовый "Сейф"



Программа для безопасного хранения текстовых секретов с использованием шифрования заменой символов.



Возможности



\- Шифрование текста заменой символов

\- Дешифрование текста обратной заменой

\- Хранение секретов в памяти

\- Простое консольное меню



Установка



1\. Клонируйте репозиторий:

git clone https://github.com/ваш-логин/text-safe.git

cd text-safe
